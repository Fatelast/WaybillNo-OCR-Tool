# 版本说明

## v0.4.0

当前版本已具备基础桌面操作流程：

- 扫描输入目录中的图片和 PDF。
- 通过 OCR 流水线识别集装箱箱号。
- 使用 ISO 6346 校验合法箱号。
- 按识别结果分类复制文件。
- 生成 `识别结果.xlsx`。
- 提供 Tkinter 桌面界面。

## 打包状态

当前已提供 `app.spec` 作为 Windows 打包配置基础，但尚未完成真实打包验收。

打包前仍需确认：

- Tesseract 已放入 `tools/tesseract`。
- Poppler 已放入 `tools/poppler`。
- 有真实样本可验证 OCR 结果。
- PyInstaller 构建产物能在无源码环境运行。

## 已知限制

- 目前尚未做真实样本端到端 OCR 验收。
- 目前尚未内置图标。
- `箱号错误` 目录为兼容输出结构预留，自动错误箱号分类仍需后续完善。
