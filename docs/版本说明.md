# 版本说明

## v0.4.0

当前版本已具备基础桌面操作流程：

- 扫描输入目录中的图片和 PDF。
- 通过 OCR 流水线识别集装箱箱号。
- 使用 ISO 6346 校验合法箱号，并要求箱号第 4 位为 `U`。
- 当整页 OCR 没有强候选时，使用重叠网格区域 OCR 和候选评分提升准确率。
- 按识别结果分类复制文件，正确识别文件会按箱号重命名。
- 生成 `识别结果.xlsx`。
- 提供 Tkinter 桌面界面。
- 提供命令行环境诊断、批量处理和样本基线验收入口。

## 打包状态

当前已提供 `app.spec` 作为 Windows 打包配置基础，但尚未完成真实打包验收。

打包前仍需确认：

- Tesseract 已放入 `tools/tesseract`。
- Poppler 已放入 `tools/poppler`。
- 有真实样本可验证 OCR 结果。
- `python -m waybill_ocr verify-samples` 能通过真实样本基线验收。
- PyInstaller 构建产物能在无源码环境运行。

## 已知限制

- 目前尚未做真实样本端到端 OCR 验收。
- 目前尚未内置图标。
