# 运单箱号识别分拣 v1.0.3

本版本提供 Windows 本地离线版程序，用于批量识别运单 PDF / 图片中的集装箱箱号，并按识别结果自动分类输出文件和 Excel 汇总表。

## 下载

请下载本次 Release 附件：

- `运单箱号识别分拣_v1.0.3_离线版.zip`

解压后双击 `运单箱号识别分拣.exe` 启动。不要把 exe 单独移出目录使用；如需移动，请移动整个解压后的文件夹。

## 主要更新

- 支持最多 2 组“输入文件夹 -> 输出文件夹”并行处理，适合批量运单分组识别。
- 输出文件夹可不选；未选择时自动在输入文件夹同级创建 `{输入文件夹名}_识别输出`。
- 增加停止处理能力，运行中锁定路径选择，避免任务处理中切换目录导致输出混乱。
- 增加速度模式：稳定模式、均衡模式、快速模式，并在界面中显示适用场景说明。
- 优化模糊运单识别：均衡/稳定模式会对高风险候选执行增强 OCR 复核，降低“错误但合法箱号”被误判为正确识别的风险。
- 快速模式增加全宽中部优先识别区域，提高常见中部箱号位置的覆盖率。
- 对未识别、箱号错误文件，如存在唯一安全复核候选，会输出为 `{候选箱号}-待确认`，方便人工验收。
- 结果 Excel 简化主表列，并对错误行使用红底标记，便于快速筛查。
- 支持预期箱号清单前置校验，并生成箱号比对工作表。
- 识别结果表被 Excel / WPS 占用时，自动写入本轮备用结果表，不中断后续处理。
- 移除识别证据截图列和证据文件夹输出，减少结果目录体积。
- 失败文件会跳过并继续处理后续文件，不再因单个文件异常中断整批任务。

## 使用说明

1. 解压 `运单箱号识别分拣_v1.0.3_离线版.zip`。
2. 双击 `运单箱号识别分拣.exe`。
3. 选择输入文件夹。
4. 输出文件夹可不选；不选时程序会自动生成输出目录。
5. 可按需选择预期箱号清单，支持 `.txt`、`.csv`、`.xlsx`。
6. 点击“开始处理”。
7. 处理完成后，在输出目录查看 `正确识别`、`未识别`、`箱号错误` 和 `识别结果.xlsx`。

## 已知限制

- 快速模式主要适合清晰文件快速粗筛；模糊文件或位置特殊的箱号可能漏识别。
- 稳定模式识别更充分，但处理速度更慢。
- 程序不会把 `O/0`、`I/1`、`B/8`、`G/6`、`T/7` 等猜测型字符替换直接作为最终正确箱号；相关信息会保留给人工复核。
- OCR 准确率仍受 PDF 清晰度、扫描质量、印章遮挡、字体粘连和页面倾斜影响。
- 请勿在处理过程中打开或锁定输出 Excel；如已打开，程序会自动写入备用结果表。
