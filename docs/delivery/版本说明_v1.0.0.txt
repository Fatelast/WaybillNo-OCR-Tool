运单箱号识别工具 版本说明 v1.0.0

本版本更新内容：
- 支持最多 2 组“输入文件夹 -> 输出文件夹”并行处理。
- 支持处理中停止任务，停止后可重新选择输入/输出路径。
- 优化 OCR 区域识别流程，优先识别高价值区域，失败后再回退完整网格。
- 支持识别失败文件跳过继续处理，避免单个文件失败中断整批任务。
- 支持结果表被占用时自动写入备用 Excel。
- Excel 中识别错误行使用红底标记，原始文件名列已加宽。
- 新增预期箱号清单比对：输出已匹配、缺失、多余识别箱号，并在日志中显示缺失箱号。

交付注意事项：
- 交付包只保留当前版本说明文件。
- exe 不要脱离依赖目录单独移动；如需移动，请移动整个交付目录。
